# Annonces

## Useful Commands 
* Connect to container:
```bash
	psql -h 10.188.41.217  -p 5432 -U postgres -W
```

* Afficher utilisateur du port :
```bash
	sudo lsof -i :5432
```

* Lancer interactive avec un container:
```bash
	docker exec -it db bash
```

* Utile jdbc:
```
	http://zetcode.com/java/postgresql/
```
